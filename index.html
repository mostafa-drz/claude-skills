Install

Code Claude Code (CLI)

  1. Clone the repo
  2. Copy skills to ~/.claude/skills/
  3. Use with /skill-name in Claude Code
git clone https://github.com/mostafa-drz/claude-skills.git
cp -r claude-skills/code/* ~/.claude/skills/

Desktop Desktop / claude.ai

  1. Download the skill folder
  2. ZIP it: zip -r skill.zip skill/
  3. Upload via Settings > Capabilities > Skills
cd claude-skills/desktop
zip -r inbox-catchup.zip inbox-catchup/
# Upload via Settings > Capabilities > Skills

Skills Catalog

Filter by platform to see what's available for your setup.

inbox-catchup Desktop

Scans all connected communication channels — Gmail, Slack, Calendar, and any available integrations — then produces a prioritized catchup briefing. Helps triage messages and draft replies.

Use when: Starting the day, returning from a break, or needing to catch up on communications.

research-assistant Desktop

Researches a topic systematically and produces a structured briefing. Gathers key facts, perspectives, and sources into a clear summary.

Use when: Researching something, preparing a briefing, or compiling background on a topic.

/address-pr-comments Code Side Effects

Fetches unresolved PR comments, categorizes them (must-fix, suggestion, question, nit), proposes fixes or replies for each, and executes approved actions.

Use when: Addressing PR review feedback or when someone requests changes on your PR.

/audit-skills Code Side Effects

Audits all personal Claude skills against the SKILLS_GUIDE.md manifest, latest official Claude skills documentation, and best practices. Reports issues, missing patterns, and improvement suggestions per skill.

Use when: Keeping skills healthy, consistent, and up-to-date with the latest standards.

/build-incremental Code Side Effects

Implements code in progressive, verified increments — auto-detects the project's toolchain, builds each unit, runs checks (typecheck, lint, test), fixes errors, and commits with semantic messages.

Use when: Building features, implementing milestones, or making multi-step changes.

/create-pr Code Side Effects

Creates a well-structured pull request with product-focused summary, change highlights, and test steps. Auto-detects base branch, links Linear issues from branch name, and pushes if needed.

Use when: Ready to open a PR or when asking to create a pull request.

/git-cleanup Code Side Effects

Identifies and removes stale git branches, orphaned remote branches, and unused worktrees. Cross-references with Linear to check issue status before deleting.

Use when: Repos have accumulated stale branches and you want to tidy up.

/investigate-ci Code

Investigates GitHub Actions workflow failures for any repo. Fetches recent runs, identifies failures, extracts error logs, diagnoses root causes, and suggests fixes.

Use when: A deploy or CI workflow fails and you need to understand why.

/organize-screenshots Code Side Effects

Scans a folder for recent screenshots, visually classifies which ones are relevant to current work, and organizes them into a target directory with descriptive filenames.

Use when: Collecting screenshots for PRs, bug reports, docs, or Linear issues.

/post-pr-for-review Code Side Effects

Generates a contextual Slack message for posting a PR to the team's review channel. Pulls context from PR diff, Linear ticket, session conversation, and related PRs.

Use when: Posting a PR for team review in Slack.

/post-ticket-summary Code Side Effects

Posts a structured implementation summary comment to a Linear issue — what was built, key decisions, reuse patterns, and how to test.

Use when: After completing work on a ticket to document the implementation for the team.

/publish-skills Code Side Effects

Publishes personal Claude skills to a GitHub repository for sharing. Copies skill files, generates a README catalog, commits, and pushes.

Use when: Ready to share skill updates or after creating/updating skills.

/shop-research Code Side Effects

Researches products across Amazon, Google Shopping, and specialty sites via the Claude-in-Chrome extension; produces a 2026 single-file HTML report with pros/cons, review highlights, and picks. Learns from feedback to personalize future searches.

Use when: Shopping for something, comparing products, finding a gift, or researching purchases before buying.

/skill-creator Code Side Effects

Creates new Claude Code skills interactively by asking contextual questions about purpose, side effects, tools, and workflow. Generates a complete SKILL.md following all conventions.

Use when: Creating a new skill or scaffolding a skill from scratch.

/slack-to-ticket Code Side Effects

Creates a Linear issue from a pasted Slack thread. Parses the conversation, infers title, priority, category, and description, checks for duplicates, and creates a clean ticket.

Use when: Pasting a Slack thread to turn it into a trackable issue.

/smoke-test Code Side Effects

Traces and verifies that something works end-to-end in any environment. Builds a check plan from natural language input, confirms it, then runs each check reporting pass/fail.

Use when: Validating deployments, pipelines, features, or migrations.

/sync-branch Code Side Effects

Merges one branch into another with conflict handling. Stashes work, updates both branches, merges, resolves conflicts preserving both sides, pushes, and restores state.

Use when: Keeping a long-lived branch in sync with its upstream.

/thread-to-action Code Side Effects

Parses a pasted thread (Slack, email, GitHub, Teams), analyzes it against current git, Linear, and session context, and suggests actionable next steps — then executes them with confirmation.

Use when: Pasting a conversation that implies developer actions.

/whats-next Code

Suggests the 3 most impactful next actions based on full developer context — git, Linear, PRs, and current conversation. Prioritizes blockers, unblocked items, and momentum.

Use when: Deciding what to work on next or after finishing a task.
