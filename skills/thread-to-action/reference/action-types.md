## Git/PR actions
- Close or merge a PR
- Update a PR description or title
- Delete a stale branch
- Cherry-pick a commit
- Rebase or update a branch
- Push changes

## Linear actions
- Update ticket status (e.g., move to "Done", "In Review", "Blocked")
- Create a new ticket for follow-up work
- Add a comment to an existing ticket
- Update ticket description with new information
- Link related issues
- Change assignee

## Code actions
- Revert a change that was superseded
- Apply a code change suggested in the thread
- Update a config or env variable

## Cleanup actions
- Close a PR that was superseded by someone else's work
- Delete a branch that's no longer needed
- Remove or archive stale tickets
